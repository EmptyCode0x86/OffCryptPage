Web Remote Control System lets you manage Windows devices directly from your browser. You can view device status in real time, send control commands, automate actions with Mouse Bot, and move files between your computer and the remote device — without being physically present.

Screen locker: Lock a Windows desktop from the web UI. The lock runs fullscreen, blocks common shortcuts, and requires a password to exit. You can trigger lock/unlock remotely and see the current state (online, locked, offline). You can also take screenshots of the locked desktop in real-time. The connection is handled by a lightweight Windows agent.

Automation: Create simple step-by-step scripts to move the mouse, click, type text, wait, and start programs. Scripts can be saved, loaded, edited, and executed on demand. You can also take screenshots of the desktop during script execution to monitor progress. Useful for repeating the same actions or assisting with routine tasks.

File Transfer: Upload and download files and browse folders on the remote device. The browser shows directory contents, supports quick paths, and offers optional encryption when sending files. Progress and results are shown in the UI.

Real‑time Status: Device state and operation results update live using SignalR. The system stack includes ASP.NET Core (API + SignalR), Blazor Server for the web UI, SQLite via EF Core for storage, and a Windows agent that executes commands.
