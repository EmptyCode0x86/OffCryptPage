Frequently Asked Questions

What is Offcode Automation Tool?

Offcode Automation Tool is a Windows desktop automation application that combines OCR text detection, UI Automation, and mouse/keyboard simulation to automate virtually any computer task.

What can I automate with this tool?

You can automate social media tasks (posting, liking, commenting), web browsing (form filling, searches), document work (Excel, Word tasks), gaming (repetitive tasks), data collection, and more.

What is the Visual Reader?

The Visual Reader is a transparent overlay that shows a scanner area (circle or rectangle) following your cursor. It continuously reads text from the selected screen region, perfect for monitoring dynamic content like game timers, chat messages, or any changing text.

How do Visual Markers work?

Visual Markers are numbered green circles you can place on the screen to mark important click positions. Choose a hotkey (e.g. 1‚Äì9, F3‚ÄìF12, Numpad) in Settings and press it to add a marker at your current mouse position. Markers display their coordinates and can be used in automation scripts for precise clicking.

How does Scheduled Start work?

Scheduled Start lets you run your automation script at specific times. Enter a date and time in the text field and click "+Add to queue" to add it to the list. You can add multiple times; they run in order. When a time is reached, the script starts automatically and that entry is removed from the list; the next scheduled time then runs when due. Use "Remove selected" to remove a chosen time from the list. Enable "Scheduled start" to activate the queue.

How do I use different text each time in my script (e.g. each loop)?

Use the Text list feature. Open it via "+Get text from file". Load lines from a .txt/.md file or add them manually. Click the button there to add a "Get text from list" step to your action sequence. When the script runs that step, the next line from the list is copied to the clipboard and marked as used. The following action can paste or type it (e.g. into a search box or form). You can choose random order so each run uses a random unused line. This way every loop iteration can use a different value from your list.

What are Scanner Triggers?

Scanner Triggers enable conditional automation using the Visual Reader. Use `[WAIT FOR TEXT] "message"` to pause your script until specific text appears in the scanner area, or `[WAIT GONE] "message"` to wait until text disappears. The script continues automatically when the condition is met.

How does OCR text detection work?

OCR (Optical Character Recognition) reads text directly from your screen, allowing the tool to find and interact with text elements like buttons, fields, and links in any application.

What is UI Automation mode?

UI Automation uses Windows' built-in accessibility features to find and interact with interface elements. It's faster and more accurate than OCR for supported applications.

How do I create an automation script?

Create an action sequence by adding steps like Wait for Text, Click, Type, Copy, Paste, and more. Use special commands like /copy:, /wait:, /type:. Press F1 to start the automation and F2 to stop.

What operating systems are supported?

Offcode Automation Tool works on Windows 10 and 11. The tool uses Windows' built-in OCR engine and accessibility features.

Does Offcode Automation Tool require coding?

No. Offcode Automation Tool is built for Windows users without programming experience. You create automations with visual actions, simple commands and an easy script editor instead of writing code.

Can I loop automation sequences?

Yes! You can set the loop count to repeat the entire sequence multiple times, or set to 0 for infinite loops (stopped with F2).

Does the software collect any data?

No, the application does not collect any personal data or usage information. All automation runs locally on your computer.

Is the license a one-time payment?

Yes, the license is a one-time payment for lifetime access. There are no recurring monthly or yearly fees.
