üìã Activation Instructions

This guide will walk you through the simple 3-step activation process to get your software up and running.

Overview

After completing your purchase, you will need to:

  1. Receive and download the Authenticator application
  2. Extract and send your unique Authentication ID
  3. Receive your Screen Lock Builder Premium software within 24 hours

Step 1: Immediate Delivery

  • When: Immediately after your purchase is completed
  • How: You will receive an email with a download link
  • Delivery Service: Selly (digital delivery platform)
  • What to Look For: Check your email inbox (and spam/junk folder) for the download link

Step 1: Download and Run

  1. Click the download link in your email
  2. Download the Authenticator.exe file
  3. System Requirements: Windows operating system
  4. Run the executable file (you may see a Windows security warning - this is normal for downloaded executables)

Step 2: Opening the Authenticator Application

  1. Launch the Authenticator.exe application
  2. The application will automatically detect your hardware components

Step 2: Locating Your Authentication ID

  • Location: The Authentication ID is displayed in the "Authentication ID:" text box
  • Security Note: The field is password-masked with asterisks (****) for security purposes

Step 2: Copying Your Authentication ID

  1. Click the "Copy Authentication ID & Email" button in the application
  2. This button will:
    • Copy both the Authentication ID and the contact email address to your clipboard
  3. You will see a success message confirming the data has been copied

Step 2: Sending Your Authentication ID

  1. Open your email client
  2. Create a new email message
  3. Paste the clipboard contents (the button has already formatted it with the email address)
  4. Send To: offcrypt.sales@gmail.com
  5. Subject: (Optional) "Screen Lock Builder Premium Activation - Authentication ID"
  6. Send the email

Important: Each Authentication ID is unique to your specific hardware configuration. Do not share this ID with others.

Step 3: Processing Time

  • When: Within 24 hours after you send your Authentication ID
  • How: Manual verification of your Authentication ID is required for security
  • Delivery Method: Email with download link

Step 3: What Happens Next

  1. Our team verifies your Authentication ID
  2. Once verified, you will receive an email with the download link
  3. The email will contain:
    • Download link for Screen Lock Builder Premium
    • Installation instructions (if needed)
    • License activation details

Step 3: If You Don't Receive the Download Link

After 24 hours: If you haven't received your download link within 24 hours, please:

  1. Check your spam/junk email folder
  2. Verify you sent the Authentication ID to the correct email address: offcrypt.sales@gmail.com
  3. Contact support at: offcrypt.sales@gmail.com
  4. Include your purchase receipt and the Authentication ID in your support request

Troubleshooting: The Authenticator Application Won't Run

  • Windows Security Warning: Click "More info" and then "Run anyway" if Windows blocks the application
  • Antivirus Software: Some antivirus software may flag the application - add it to exceptions if needed
  • Administrator Rights: Try running the application with administrator privileges
  • Corrupted Download: Re-download the file from the original email link

Troubleshooting: I Haven't Received My Download Link

  • Check Spam Folder: Sometimes delivery emails end up in spam/junk folders
  • Wait 24 Hours: Processing can take up to 24 hours
  • Verify Email Sent: Double-check that you sent the Authentication ID to offcrypt.sales@gmail.com
  • Contact Support: Email offcrypt.sales@gmail.com with:
    • Your purchase receipt
    • Your Authentication ID
    • Purchase date and time
