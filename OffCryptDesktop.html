OffCrypt Desktop - Professional Windows Encryption with X25519 and ECDH+HKDF+AES-GCM

OffCrypt Desktop is a professional Windows encryption application providing secure text and file encryption using cutting-edge cryptographic algorithms. Features both password-based encryption (ECDH+HKDF+AES-GCM) and X25519 public-key cryptography with NIST 2024 compliance for maximum security.

This is a completely offline encryption application - no internet connection required or used. All cryptographic operations happen locally on your Windows system with zero data collection. Perfect for professionals, journalists, and privacy-conscious users who need secure communication without cloud dependencies.

Advanced features include disappearing messages with configurable expiration times, PGP-style identity management with contact trust levels, multiple ECDH curve options, and comprehensive file encryption with .enc and .pgp format support. Built for Windows with secure memory management and professional-grade security features.

Set Master Password

Go to Tab 1, enter a strong password (8+ characters) and click "Set Password". The password is stored securely in memory as char[] array.

Type your message in the large text area. Optionally configure disappearing messages, then click "Encrypt" to create encrypted text.

3

Share Encrypted Text

The encrypted message is automatically copied to clipboard. Share it through any channel - email, chat, or save as .enc file.

4

Decrypt Messages

Copy encrypted text to clipboard, enter the password, and click "Decrypt" to reveal the original message.
