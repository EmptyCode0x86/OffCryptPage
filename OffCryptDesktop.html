💻 OffCrypt - Desktop

Open-source

OffCrypt Desktop is Windows encryption application providing secure text and file encryption using cutting-edge cryptographic algorithms. Features both password-based encryption (ECDH+HKDF+AES-GCM) and X25519 public-key cryptography with NIST 2024 compliance for maximum security.

This is a completely offline encryption application - no internet connection required or used. All cryptographic operations happen locally on your Windows system with zero data collection. Perfect for professionals, journalists, and privacy-conscious users who need secure communication without cloud dependencies.

Advanced features include disappearing messages with configurable expiration times, PGP-style identity management with contact trust levels, multiple ECDH curve options, and comprehensive file encryption with .enc and .pgp format support. Built for Windows with secure memory management and professional-grade security features.

🎯 How to Use

Step-by-step guide to secure encryption

🔐

Password-Based Encryption

Simple and secure password-protected messages

1

Set Master Password

Enter a strong password (8+ characters) and click "Set Password". Stored securely in memory.

2

Encrypt Message

Type your message, configure disappearing options, then click "Encrypt" to create secure text.

3

Share & Decrypt

Message auto-copies to clipboard. Share via any channel. Recipients decrypt with the same password.

🔑

X25519 Public Key

PGP-style secure communication without servers

1

Generate Keys

Click "Generate Keys" to create your X25519 key pair. Keys protected with Windows DPAPI.

2

Share Public Key

Copy and share your public key with anyone who wants to send you encrypted messages.

3

Import & Communicate

Import recipient's public key, then send secure messages only they can read.
